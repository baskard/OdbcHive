■導入方法
---------------------------------------------------------------------------

　「Hive ODBC driver for Windows」のコンパイルにはVisual Studioが必要です。

(1)ソース一式のダウンロード
(2)ダウンロードしたファイルを展開
(3)Visual Studioコマンドツールを起動
(4)展開先ディレクトリでコンパイル
   > cd <展開先ディレクトリ>
   > nmake
(5)OdbcHiveのレジストリ登録
　環境にあわせてhiveodbc.regを編集
　hiveodbc.regをダブルクリック
